This app checks whether today is a public holiday in South Korea.

It helps you automate your home based on holidays. For example, you can prevent your wake-up alarm from ringing on a holiday morning or keep the heating on during the day.

Features:
- Check if today is a Korean public holiday.
- Use flow cards to trigger actions based on holiday status.
- Real-time updates using public data.

Note:
This app requires an internet connection to fetch the latest holiday data.
